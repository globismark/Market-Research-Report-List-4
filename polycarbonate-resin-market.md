<p><p>Polycarbonate resin can be categorized into two main types based on the method of production: Phosgene type and Non-Phosgene type. Phosgene type polycarbonate resin is produced using a phosgene-based method, which involves reacting bisphenol A with phosgene. On the other hand, Non-Phosgene type polycarbonate resin is produced using a non-phosgene process, which typically involves reacting dimethyl carbonate with bisphenol A. Both types of polycarbonate resin have their own set of advantages and applications in various industries.</p></p>
